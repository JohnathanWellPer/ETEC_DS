## 📚 Sobre o Repositório

Este repositório funciona como um portfólio acadêmico e um registro de evolução técnica. Ele espelha a estrutura curricular de 1200 horas do curso de Desenvolvimento de Sistemas da ETEC, focando em aplicações do mundo real, desde a lógica fundacional até a implementação de sistemas multicamadas, aplicativos móveis e comunicação com banco de dados.

O objetivo de manter estes códigos organizados é facilitar a consulta rápida de sintaxes aprendidas, documentar o progresso na escrita de software e comprovar a experiência prática adquirida durante as etapas de qualificação e habilitação profissional.

## 📂 Estrutura de Diretórios e Módulos

Os arquivos estão divididos rigorosamente conforme a matriz curricular oficial da instituição.

### 📁 [Módulo 1](./modulo_1) (Auxiliar de Desenvolvimento de Sistemas)
Foco na construção da base lógica e visual.
* **Lógica de Programação:** Construção de algoritmos, estruturas de decisão e repetição.
* **Web Design Estático:** Marcação com HTML5, estilização com CSS3 e introdução a scripts do lado cliente com JavaScript e jQuery.
* **Modelagem de Dados:** Criação de modelos conceituais e lógicos para banco de dados.
* **Infraestrutura Básica:** Sistemas operacionais, linha de comando e noções de redes de computadores.

### 📁 [Módulo 2](./modulo_2) (Programador de Computadores)
Foco na programação orientada a objetos e na integração de sistemas.
* **Desenvolvimento Web e Desktop:** Conexão de aplicações com bancos de dados e interfaces gráficas.
* **Linguagem de Servidor:** Introdução ao PHP para rotinas dinâmicas.
* **Banco de Dados Relacional:** Linguagem SQL para criação de tabelas, consultas e manipulação de dados (DML e DDL).
* **Engenharia de Software:** Levantamento de requisitos, modelagem com UML e metodologias ágeis (Scrum).

### 📁 [Módulo 3](./modulo_3) (Técnico em Desenvolvimento de Sistemas)
Foco em especialização, mobilidade e segurança da informação.
* **Desenvolvimento Mobile:** Codificação de aplicativos para Android utilizando consumo de APIs e recursos de hardware (sensores e localização).
* **Sistemas Embarcados:** Programação em C/C++ aplicada a microcontroladores como Arduino.
* **Segurança e Qualidade:** Criptografia, prevenção contra injeção de SQL, engenharia social e rotinas de testes de software.
* **Projeto Final (TCC):** Estruturação do Trabalho de Conclusão de Curso aplicando todos os conceitos práticos adquiridos.
