# Meus Sites Favoritos - Agenda 12 🌐

Este projeto foi desenvolvido como atividade prática da **Agenda 12** do curso técnico em **Desenvolvimento de Sistemas (ETEC)**. O objetivo foi criar uma página web estruturada em HTML para listar e organizar links de sites favoritos.

## 🚀 Funcionalidades
- Página estruturada com títulos hierárquicos (`<h1>`).
- Exibição de imagens locais dos sites favoritos.
- Links direcionais para acesso rápido aos serviços.
- Imagens clicáveis (recurso opcional implementado).

## 🛠️ Tecnologias e Conceitos Utilizados
- **Linguagem:** HTML5 (HyperText Markup Language)
- **Conceitos Praticados:**
  - Estruturação de documentos web.
  - Uso de tags de texto (`<h1>`, `<p>`).
  - Inserção de mídia (`<img>`).
  - Criação de hiperlinks (`<a>`).
  - Organização de diretórios e caminhos de arquivos.

## 📌 Sites Listados
1. **YouTube** - Plataforma de vídeos.
2. **Instagram** - Rede social visual.
3. **Gemini** - Inteligência artificial do Google.
4. **Shopee** - Marketplace e e-commerce.
5. **Twitch** - Plataforma de streaming.

## 📋 Como visualizar
Para visualizar o projeto localmente:
1. Baixe todos os arquivos do repositório.
2. Certifique-se de que as imagens (`site1.jpg` a `site5.jpg`) estão na mesma pasta do arquivo HTML.
3. Abra o arquivo `JohnathanPereira_Ag12_DSI.html` em qualquer navegador.
