# Sistema de Monitoramento de Reservatório de Água 🌊

Este projeto foi desenvolvido como parte das atividades da **Agenda 11** do curso técnico em **Desenvolvimento de Sistemas (ETEC)**. O sistema simula o monitoramento de um reservatório de água, exibindo alertas visuais coloridos no terminal de acordo com o nível de ocupação.

## 🚀 Funcionalidades
- Exibição de 5 níveis de alerta progressivos.
- Interface amigável via terminal utilizando cores para facilitar a identificação de riscos.
- Tratamento automático de estilos do terminal (limpeza de cores após execução).

## 🛠️ Tecnologias e Conceitos Utilizados
- **Linguagem:** Python 3
- **Biblioteca:** [Colorama](https://pypi.org/project/colorama/) (para estilização do terminal).
- **Conceitos de Programação:**
  - Listas para armazenamento de dados (Agenda 9).
  - Funções para modularização e organização do código (Agenda 10).
  - Estruturas de repetição (`for`) e condicionais (`if/elif`).
