# 📊 Sistema de Pesquisa de Satisfação - TudoWeb

Este é um projeto desenvolvido em **Python** para automatizar a coleta e tabulação de pesquisas de opinião de clientes.

## 🚀 O que o projeto faz?
O sistema simula o atendimento a múltiplos clientes utilizando estruturas de repetição e decisão. Ele coleta a opinião sobre o atendimento e, ao final do ciclo, exibe um painel gerencial com os resultados tabulados.

## 🛠️ Tecnologias e Conceitos Utilizados
* **Linguagem:** Python
* **Estrutura de Repetição:** Laço `for` com `range()` para automatizar a amostragem.
* **Estrutura de Decisão:** `if` e `elif` para categorizar as respostas e alimentar os contadores.
* **Operadores de Atribuição:** Utilização de incrementos (`+= 1`) para contagem de dados.

## 💡 Como executar
1. Tenha o Python instalado na máquina.
2. Clone este repositório.
3. Rode o comando `python JohnathanPereira_Ag8_DS_I.py` no seu terminal.
